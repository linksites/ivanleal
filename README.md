# Ivan Leal JJ — Site institucional

Projeto em Next.js + TypeScript para o site institucional do mestre e atleta Ivan Leal JJ.

## Objetivo

Criar uma presença digital premium, moderna e real, com galeria, vídeos, conquistas, seminários e CTA para contato.

## Rodar localmente

```bash
npm install
npm run dev
```

Depois acesse:

```bash
http://localhost:3000
```

## Estrutura

- `app/`: rotas do site
- `components/`: blocos reutilizáveis da interface
- `data/`: textos e dados editáveis
- `public/images/`: imagens do mestre Ivan Leal
- `lib/`: SEO, analytics e utilidades
